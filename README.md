# office-bty-pages

这是一个用于归档和发布多个独立 HTML 页面的仓库。

## 项目简介

本仓库主要用于存放各类独立制作的 HTML 页面文件。这些页面彼此独立，不依赖特定的后端服务或域名环境，适合作为静态页面存档或临时展示用途。

## 目录说明

```
├── pages/          # 存放所有 HTML 页面文件
├── assets/         # 公共资源文件（如 CSS、JS、图片等）
├── index.html      # 仓库页面索引（可选）
└── README.md       # 本文件
```

- `pages/`：每个子目录或文件代表一个独立的页面项目，命名清晰即可。
- `assets/`：如果页面间有共享资源，可统一放置在此目录下。
- `index.html`：可选的首页索引，用于快速浏览所有已归档页面。

## 页面归档说明

- 每个页面均为完整的 HTML 文件，可直接在浏览器中打开运行。
- 页面之间无耦合关系，可独立更新或删除。
- 归档时请保持文件结构整洁，避免冗余或未使用的资源文件。

## 维护说明

- 本仓库由维护者不定期更新，新增页面会按需添加至 `pages/` 目录。
- 欢迎提交 Issue 或 Pull Request 来建议新增或改进现有页面。
- 请确保提交的内容不包含敏感信息或违反相关法律法规。
- 所有页面仅供归档展示，不提供任何形式的服务承诺或技术支持。

## 使用方式

1. 克隆本仓库到本地：
   ```bash
   git clone https://github.com/your-username/office-bty-pages.git
   ```
2. 直接在浏览器中打开 `pages/` 目录下的 HTML 文件即可查看。
